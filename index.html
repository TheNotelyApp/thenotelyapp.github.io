NotelyWrite. Organize. Remember.

A desktop Markdown notes app with Git version control, AI assistance, and offline-first sync.

💻 Download for Windows
🚀 Getting Started
📚 All Documentation
📝

Markdown Editor

Edit, Split, and Preview modes. Toolbar shortcuts, inline table editor, code block formatting, Mermaid and Excalidraw diagrams.

🌿

Git Version Control

Native Git integration. Commit, browse history, compare diffs, restore notes, manage branches — all from inside the app.

🤖

AI Writing Assistant

AI chat, AI palette actions, semantic search, and relationship graph powered by your choice of AI provider.

🔍

Powerful Search

Global search across all notes with regex support, code-block filtering, and meaning-based results when AI is configured.

🔒

Works Offline

All core features work without internet. Your notes stay on your device in plain Markdown files.

🔄

P2P Sync

Sync notes across devices over your local network using encrypted peer-to-peer pairing — no cloud required.

📦

Export / Import Note Packages

Bundle notes with all linked assets into an encrypted, integrity-checked `.note` file. Share with others and import directly into any Notely workspace.
